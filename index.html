  • Developed background and foreground services for continuous data collection and task execution.
  • Integrated Google Maps for live tracking, rescue coordination, and visualization of affected areas.
  • Built push notification systems and role-based access control to support multi-user interaction (rescue teams, admins, and analysts).

  • Utilized Dio for API integration and SQLite / Secure Storage for local persistence.
